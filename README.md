# Cernunnos - Personal Finance Cash Flow Visualization Tool

A monorepo personal finance app for importing bank transactions (Chase checking & credit card CSVs) and visualizing cash flow with D3.js.

## Tech Stack

- **Frontend:** Vite + TypeScript + D3.js
- **Backend:** Node/Express + TypeScript
- **Database:** SQLite (better-sqlite3)

## Getting Started

```bash
npm install
npm run dev
```

- Client: http://localhost:5173
- Server: http://localhost:3001

## Project Structure

```
client/   — Vite + TypeScript frontend
server/   — Express + TypeScript backend
test-data/ — Sample CSV files for testing
```
